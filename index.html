A domain without a blank page is always nice.

Original content before migration to static gitpages
This website is is a work in progress, at the moment it is a simple cms (named µCMS v1.0) written from scratch in php with a simple basic design and some URL rewriting to keep linking tidy.

Some javascript has been added to give the pages some life for now with random fading background colours which you can stop/start using the controls at the bottom of the page. If you have visited the web site before you will notice the background colour will be restored to what it was when you were last here...more contents, features, additions are coming soon!

You can contact me via a social site: twitter, etc

For those that are interested (read on) - as technical exercise all components are compressed and javascript packed for faster page loads. Yes, it's not needed for a tiny website like this BUT it's a shared host so everything helps :-)
